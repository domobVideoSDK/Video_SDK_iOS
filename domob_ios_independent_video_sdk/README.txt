---------------------------------
  多盟 iOS 视频积分墙 SDK 发行包，说明文档
---------------------------------

本SDK包的内容如下: 

IndependentVideoSDK\
    包含 Domob iOS 视频积分墙 SDK 库文件、头文件以及相关支持文件

Samples\
    示例工程。
    为了在运行示例工程时能出广告,请从多盟官网网站（http://www.domob.cn）的积分墙板块获取您的Publisher ID。

UserGuide.pdf
    Domob积分墙官网用户使用指南

SDKUserGuide.pdf
    Domob iOS 视频积分墙 SDK 的用户指南文档

README.txt
    本自述文件
